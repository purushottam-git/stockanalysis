### 1. Stock Analysis
- Real-time data fetching using **Yahoo Finance**.

### 2. AI Stock Prediction
- Forecasts stock prices for the next **30 days**.
- Uses **ARIMA** (AutoRegressive Integrated Moving Average) machine learning model.
- Visualizes the trajectory of Historical Data vs. AI Forecast.
- Displays Model Accuracy (RMSE) and Potential Growth %.

### 3. CAPM Analysis (Capital Asset Pricing Model)
- **CAPM Return:** Calculates expected return based on market risk.
- **CAPM Beta:** Calculates individual stock volatility relative to S&P 500.
- Compares assets against the market benchmark.
